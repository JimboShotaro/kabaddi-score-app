# ストア掲載用画像素材（v1）

Google Play Consoleへの掲載に必要な画像素材を配置するフォルダです。
以下のサイズの画像を用意して、このフォルダに保存してください。

## 必須素材

1.  **アプリアイコン**
    *   サイズ: 512 x 512 px
    *   形式: 32ビット PNG
    *   容量: 1MB以下

2.  **フィーチャーグラフィック**（ストアのトップに表示される横長画像）
    *   サイズ: 1024 x 500 px
    *   形式: JPG または 24ビット PNG（アルファなし）
    *   容量: 1MB以下

3.  **スクリーンショット（スマートフォン）**
    *   アスペクト比: 16:9 または 9:16
    *   各辺のサイズ: 320px 〜 3840px
    *   枚数: 2枚以上（最大8枚）
    *   形式: JPG または 24ビット PNG

## 推奨素材（任意）

4.  **スクリーンショット（7インチタブレット）**
    *   枚数: 4枚以上推奨

5.  **スクリーンショット（10インチタブレット）**
    *   枚数: 4枚以上推奨
