# プライバシーポリシー（雛形）

本アプリ「カバディーアプリ」（以下、「当アプリ」といいます）における利用者情報の取り扱いについて、以下のとおりプライバシーポリシー（以下、「本ポリシー」といいます）を定めます。

## 1. 収集する情報

当アプリでは、以下の情報を収集する場合があります。
（※アプリの実装に合わせて選択・修正してください）

*   **端末情報**: アプリの不具合解析や改善のため、端末のモデル名、OSバージョンなどの情報を収集することがあります。
*   **利用状況データ**: アプリの利用状況（画面の閲覧履歴、ボタンのタップ数など）を収集することがあります。
*   **広告ID**: 広告配信のために、広告ID（Advertising ID / IDFA）を利用することがあります。

（※個人情報を収集しない場合は、「当アプリは、利用者の個人情報を収集しません。」と記載します）

## 2. 利用目的

収集した情報は、以下の目的で利用します。

1.  アプリの機能提供および維持・改善のため
2.  不具合の調査・修正のため
3.  利用状況の分析によるサービス向上のため
4.  （広告を表示する場合）適切な広告を配信するため

## 3. 第三者への提供

当アプリは、法令に基づく場合を除き、利用者の同意なく第三者に個人情報を提供することはありません。
ただし、以下の情報収集モジュールを利用する場合、各提供者のプライバシーポリシーに従って情報が送信されることがあります。

*   **Google AdMob / Firebase Analytics**
    *   提供者: Google LLC
    *   利用目的: 広告配信、利用状況解析
    *   プライバシーポリシー: https://policies.google.com/privacy

## 4. 免責事項

当アプリの利用により生じた損害について、開発者は一切の責任を負わないものとします。

## 5. プライバシーポリシーの変更

本ポリシーの内容は、法令その他本ポリシーに別段の定めのある事項を除いて、利用者に通知することなく変更することができるものとします。
変更後のプライバシーポリシーは、本ページに掲載したときから効力を生じるものとします。
